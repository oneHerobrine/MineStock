# MineStock 1.2.1 更新说明

## 新增功能

### 港股市场支持

本版本正式加入对香港股票市场的支持，玩家现在可以查询、买卖港股。

**港股代码格式**：1-5 位纯数字，例如：
- `700` — 腾讯控股
- `9988` — 阿里巴巴
- `3690` — 美团
- `1810` — 小米集团

数据来源同 A 股，使用新浪财经（主）和腾讯财经（备），无需配置 API Key。

**三大市场汇总：**

| 市场 | 代码格式 | 示例 | 数据源 | 需要 API Key |
|------|----------|------|--------|--------------|
| A 股 | 6 位数字 | `000001`、`600519` | 新浪 / 腾讯备用 | 否 |
| 港股 | 1-5 位数字 | `700`、`9988` | 新浪 / 腾讯备用 | 否 |
| 美股 | 1-10 位字母 | `AAPL`、`TSLA` | Finnhub / Twelve Data 备用 | 是 |

### 新增命令别名

除原有的 `/mstock` 外，本版本新增了两个命令入口：

| 命令 | 类型 | 说明 |
|------|------|------|
| `/mstock` | 静态 | 原主命令，始终可用 |
| `/stock` | 静态 | 新增静态别名，始终可用 |
| `/st` | 动态 | 启动时自动检测是否冲突，无冲突则注册 |

`/st` 为动态注册——插件启动时若检测到该命令已被其他插件占用，则自动跳过注册并在控制台输出提示，不影响其余功能正常运行。

三个命令完全等价，使用相同的权限（`minestock.use`）和子命令体系。

---

## 使用示例

```
# 查看港股行情 + K 线
/stock 700
/st 9988

# 买入港股
/mstock buy 700 100

# 卖出港股
/stock sell 3690 50

# 查看持仓（A股/港股/美股混合显示）
/st portfolio

# 查看推荐（需在 recommended-pool 中加入港股代码）
/mstock recommended
```

---

## 配置说明

`config.yml` 中可将港股代码加入推荐池：

```yaml
recommended-pool:
  - "000001"   # A 股
  - "600519"   # A 股
  - "700"      # 港股 — 腾讯控股
  - "9988"     # 港股 — 阿里巴巴
  - "AAPL"     # 美股
  - "TSLA"     # 美股
```

港股无需额外配置项，K 线缓存、价格倍率、手续费等全局配置同样适用于港股。
